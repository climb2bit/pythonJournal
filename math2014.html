1 The Double-Counted Sum
Problem

John summed the numbers 1 to n, but accidentally added one number twice, arriving at a total of 3405. Find the number he added twice.

Key Idea

The true sum 1+2+…+n = n(n+1)/2. Adding one extra copy of x gives the wrong total, so:

Wrong total = n(n+1)/2 + x = 3405 We need: n(n+1)/2 < 3405 < n(n+1)/2 + n (because 1 ≤ x ≤ n)
Working
  1. Test n = 82:   82 × 83 / 2 = 3403.
  2. Remainder:   3405 − 3403 = 2.   And 2 ≤ 82 ✓
  3. Test n = 81:   81 × 82 / 2 = 3321, remainder = 84 > 81 ✗
  4. Test n = 83:   83 × 84 / 2 = 3486 > 3405 ✗ (impossible)
  5. So n = 82 is the only valid choice, and x = 2.
Answer The number added twice is 2
2 Tilted Squares on a 5 × 4 Dot Grid
Problem

Using dots arranged in a 5 × 4 grid as vertices, how many squares can be formed such that none of the sides are parallel to the edges of the grid?

Key Idea

A tilted square is defined by a vector (a, b) with a, b ≥ 1. Its four vertices are obtained by rotating this vector 90° each time. The square fits inside a bounding box of size (a+b) × (a+b).

Counting positions: In a grid of 5 columns × 4 rows, the number of positions a bounding box of side s can sit is (5−s) × (4−s).
All valid (a, b) pairs with a, b ≥ 1
(a,b) → s = a+b → positions = (5−s)×(4−s) (1,1) → s=2 → 3 × 2 = 6 (1,2) → s=3 → 2 × 1 = 2 (2,1) → s=3 → 2 × 1 = 2 (different tilt from (1,2)) (1,3) → s=4 → 1 × 0 = 0 (3,1) → s=4 → 1 × 0 = 0 (2,2) → s=4 → 1 × 0 = 0 Note: 5×4 grid means s can be at most 3 for any positions to exist. Total = 6 + 2 + 2 = 10
Visual — examples of each tilted type
Each (a,b) pair defines a unique family of squares. (1,1): side = √2 → 6 squares (1,2): side = √5 → 2 squares (2,1): side = √5 → 2 squares Total = 10
Answer 10 tilted squares
3 Standard Young Tableaux
Problem

In each figure, numbers increase left-to-right in every row and top-to-bottom in every column. Count the number of valid fillings for each shape.

Key Idea — Hook Length Formula

The number of standard Young tableaux for a shape is given by:

Number of ways = n! / (product of all hook lengths) Hook length of cell (i,j) = (cells to the right) + (cells below) + 1
Verification for Figure 1 (2×2): hooks are 3, 2, 2, 1 → product = 12.   4!/12 = 2. ✓

5 Minimum Area of Triangle DEF
Problem

Rectangle ABCD with AB = 40, BC = 50. Point E on AB, point F on BC, with BE + BF = 30. Find the minimum area of triangle DEF.

Setup — Coordinate System
Place B = (0, 0) A = (0, 40), C = (50, 0), D = (50, 40) E on AB: E = (0, e), BE = e F on BC: F = (f, 0), BF = f Constraint: e + f = 30, 0 ≤ e ≤ 30
Area Formula
Area(DEF) with D=(50,40), E=(0,e), F=(f,0): = ½ |x_D(y_E − y_F) + x_E(y_F − y_D) + x_F(y_D − y_E)| = ½ |50e + 0·(−40) + f(40 − e)| = ½ |50e + 40f − ef| Substitute f = 30 − e: = ½ |50e + 40(30−e) − e(30−e)| = ½ |50e + 1200 − 40e − 30e + e²| = ½ (e² − 20e + 1200)
Minimisation
Minimise g(e) = e² − 20e + 1200 g'(e) = 2e − 20 = 0 → e = 10 (discriminant of e²−20e+1200 is 400−4800 < 0, so g(e) > 0 always — no sign issue) g(10) = 100 − 200 + 1200 = 1100 Min Area = 1100 / 2 = 550
Optimal position: BE = 10, BF = 20. The triangle DEF then has minimum area 550 square units.
Minimum Area 550 square units
